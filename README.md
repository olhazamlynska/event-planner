# 📖 <a name="about-project"></a>

**Event Planner** is a simple web application built with React. The app allows
users to create, edit, and view detailed information about various events.

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Create Events:** Users can easily create new events by providing event
  details such as the event name, date, time, location, and description.
- **Edit Events:** Existing events can be edited and updated with new
  information. Users can modify event details whenever necessary.
- **View Event Information:** The app provides a clear and organized view of all
  events. Users can see the event name, date, and location at a glance, and by
  clicking on an event, they can view the full event details, including the
  description.

## 💻 Getting Started <a name="getting-started"></a>

### Setup

Clone this repository to your desired folder:

- [ ] Navigate to the directory where you want to download the project.
- [ ] Or open a terminal in this directory and run the command
      `git clone https://github.com/olhazamlynska/event-planner`

### Install

Change into the projectdirectory: `cd event-planner`, then:

- [ ] Install the dependencies: `npm install`

### Usage

To run the project, execute the following command:

- [ ] Start the development server: `npm start`
- [ ] Open your browser and visit `http://localhost:3000` to view the
      application.
